# Security Policy

## Reporting Vulnerabilities
If you discover a vulnerability within this project, please report it by sending an email to security@somniascope.org. Your report will be reviewed and we will work on a fix as soon as possible.

## Best Practices for Users and Developers
- Keep your dependencies up to date.
- Follow security best practices when using your tools and libraries.
- Regularly review your code for vulnerabilities.

## Supported Versions
We support the current version of this project and the two prior versions. Please ensure you're using a supported version to receive timely security updates and fixes.

## Contact Information
For any questions or suggestions related to security, please contact us via our support email: support@somniascope.org.

This security policy is subject to change, and will be updated periodically to reflect any new best practices, contact information, or changes in supported versions.
